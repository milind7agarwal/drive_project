##Drive Project

A cloud storage web application that allows users to upload, manage, and organize files securely.

---

##  Features

-  User Authentication (Login / Signup)
-  File Upload
-  File Management (View / Delete / Download)
-  Cloud Storage System
-  Clean UI

---

##  Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB
